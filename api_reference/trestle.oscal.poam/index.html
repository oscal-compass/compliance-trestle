    Attributes
    description: str pydantic-field

    A key word to indicate all.

    include_subjects: ConstrainedListValue pydantic-field
    links: ConstrainedListValue pydantic-field
    props: ConstrainedListValue pydantic-field
    remarks: Remarks pydantic-field
    type: str pydantic-field required

    Indicates the type of assessment subject, such as a component, inventory, item, location, or party represented by this selection statement.

    Attributes

    props: ConstrainedListValue pydantic-field
    remarks: Remarks pydantic-field
    sources: ConstrainedListValue pydantic-field required
    uuid: ConstrainedStrValue pydantic-field required

    Uniquely identifies a set of assessment subjects that will be identified by a task or an activity that is part of a task.

    subject_placeholder: AssessmentSubjectPlaceholder pydantic-field
    subjects: ConstrainedListValue pydantic-field

    Attributes

    Entry pydantic-model

    Attributes
    description: str pydantic-field

    A human-readable description of this event.

    end: datetime pydantic-field

    Identifies the end date and time of an event. If the event is a point in time, the start and end will be the same date and time.

    links: ConstrainedListValue pydantic-field
    logged_by: ConstrainedListValue pydantic-field
    props: ConstrainedListValue pydantic-field
    related_responses: ConstrainedListValue pydantic-field
    remarks: Remarks pydantic-field
    start: datetime pydantic-field required

    Identifies the start date and time of an event.

    status_change: RiskStatus pydantic-field
    title: str pydantic-field

    The title for this event.

    uuid: ConstrainedStrValue pydantic-field required

    Uniquely identifies an assessment event. This UUID may be referenced elsewhere in an OSCAL document when referring to this information. A UUID should be consistently used for this schedule across revisions of the document.

    Facet pydantic-model

    Attributes
    links: ConstrainedListValue pydantic-field
    name: str pydantic-field required

    The name of the risk metric within the specified system.

    props: ConstrainedListValue pydantic-field
    remarks: Remarks pydantic-field

    value: str pydantic-field required

    Indicates the value of the facet.
