    Address pydantic-model

    Attributes
    addr_lines: ConstrainedListValue pydantic-field

    City, town or geographical region for the mailing address.

    The ISO 3166-1 alpha-2 country code for the mailing address.

    postal_code: str pydantic-field

    Postal or ZIP code for mailing address

    state: str pydantic-field

    State, province or analogous geographical region for mailing address

    type: str pydantic-field

    Indicates the type of address.

    Attributes
    __root__: str pydantic-field required special

    A single line of an address.

    links: ConstrainedListValue pydantic-field
    props: ConstrainedListValue pydantic-field
    remarks: str pydantic-field

    Commentary about the system's authorization boundary that enhances the diagram.

    Attributes

    Attributes

    Specifies a media type as defined by the Internet Assigned Numbers Authority (IANA) Media Types Registry.

    value: str pydantic-field required

    ByComponent pydantic-model

    Attributes
    description: str pydantic-field required

    An implementation statement that describes how a control or a control statement is implemented within the referenced system component.

    export: Export pydantic-field

    Identifies content intended for external consumption, such as with leveraged organizations.

    implementation_status: ImplementationStatus pydantic-field
    inherited: ConstrainedListValue pydantic-field
    links: ConstrainedListValue pydantic-field
    parameter_settings: Dict[str, trestle.oscal.ssp.SetParameter] pydantic-field
    props: ConstrainedListValue pydantic-field
    remarks: Remarks pydantic-field
    responsible_roles: Dict[str, trestle.oscal.ssp.ResponsibleRole] pydantic-field
    satisfied: ConstrainedListValue pydantic-field
    uuid: ConstrainedStrValue pydantic-field required

    A globally unique identifier that can be used to reference this by-component entry elsewhere in an OSCAL document. A UUID should be consistently used for a given resource across revisions of the document.
